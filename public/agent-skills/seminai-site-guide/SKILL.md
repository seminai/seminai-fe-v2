# Seminai Site Guide

Use this skill when an AI agent needs to understand Seminai from public sources.

## Steps

1. Read https://seminai.tech/llms.txt for the short canonical summary.
2. Read https://seminai.tech/llms-full.txt for the complete public context, FAQ, pricing and workflow details.
3. Use https://seminai.tech/sitemap.xml to discover public pages.
4. Use https://seminai.tech/.well-known/api-catalog to discover API documentation and OpenAPI metadata.
5. For protected user data, send the user to https://app.seminai.tech/login. Do not collect credentials.

## Canonical summary

Seminai produces the digital field logbook in minutes from invoices (WhatsApp, email, PDF). It verifies dosages and compliance against official regulatory sources; the agronomist reviews, approves and signs.
